Compares a manuscript text against a comparison text -- can be used for brit-picking or testing historical fiction for
anachronistic language, just fill the file "comparison_text.txt" with the the text you want to
compare your manuscript to.

0) download and set-up PyCharm,
1) download the code from GitHub,
2) open the folder with the code in it using PyCharm,
3) copy-paste your manuscript into "manuscript.txt",
4) spend some quality time on Project Gutenberg, pulling your comparison texts and pasting them into "comparison_text.txt"
remember, if you change the name of a file, use "Refactor" to make sure it updates in the code),
5) run it and it'll pop out a list like this:

0: asteroid
1: iPhone
2: yeet
